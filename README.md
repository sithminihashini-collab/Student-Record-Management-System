# Student Record Management System

## Description

This is a console-based Java application developed using Java.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Sort Students
- Registration Queue
- Undo Delete

## Data Structures

- ArrayList
- Queue
- Stack

## Algorithms

- Linear Search
- Bubble Sort

## Software

- Java
- VS Code
- JDK 21
